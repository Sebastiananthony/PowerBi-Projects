# PowerBi-Projects
The project is based on an E-commerce data set for a retail store in the United States.
The data used for the project was pulled from Kaggle and required some data cleaning, 
modelling and learning some new dax functions as well as proper application of some visual formatting to complete this project.
PROBLEM STATEMENT:
1.Create a KPI Banner showing YTD(Year to date) sales, YTD profit, YTD quantity sold, YTD profit margin.
2. Find Year on Year growth for each KPI and show a YTD sparkline for each measure in the KPI to understand the monthly trend of each fact.
3. Find YTD sales, PYTD(previous year to date) sales YOY(year on year) sales growth for different category, 
and add a trend icon (upward or downwards) for each category.
4. Find YTD sales performanceby each state.
5. Top 5 and bottom 5 products by sales.
6. YTD sales by region to know best and worst performing region all over the country.
7. YTD sales by shipping type and best shipping type by percentage.
